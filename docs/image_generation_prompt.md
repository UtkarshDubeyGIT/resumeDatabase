# ChatGPT / DALL-E Image Generation Prompt

You can copy and paste the prompt below directly into ChatGPT (or Midjourney) to generate concept art, lifestyle illustrations, or UI mockups of a user interacting with the Resumint platform. 

It specifically highlights the AI-assisted selection workflow and the modern design system we established.

***

**Prompt for Image Generation:**

> "Create a high-quality, modern UI/UX lifestyle illustration showing a young professional sitting at a clean, minimalist desk, using a sleek web application called 'Resumint' on a large computer monitor. 
> 
> The screen on the monitor should clearly display a beautiful, modern dashboard with a light gray background and floating white cards with soft drop shadows. The UI features a premium color palette of vibrant blue (#5A3FFF), light purple (#7C4DFF), and emerald green (#00B894) for success badges. 
> 
> The core workflow is visible on the screen: on the left side, there is a 'Job Description' input panel. On the right side, an AI assistant section is suggesting 'Recommended Projects' and 'Recommended Skills' with perfectly formatted, ATS-friendly bullet points. There are checkmarks next to these bullet points, and the user is actively selecting which points to include in their tailored resume. 
> 
> The overall aesthetic should feel premium, smart, and effortless, featuring modern typography, subtle glassmorphism UI elements, and a friendly, confident vibe. The user looks focused but relaxed and happy, indicating how easy the platform makes it to tailor a resume. The art style should be a clean, modern, vibrant 3D digital illustration typical of top-tier SaaS landing pages."
